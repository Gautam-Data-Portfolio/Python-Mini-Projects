###### **🐍 Snake Water Gun Game (Python Mini Project)**



* A simple command-line based Snake 🐍 Water 🌊 Gun 🔫 game built using Python.
* This project simulates a classic game similar to Rock–Paper–Scissors, where the user plays against the system.



* The system randomly selects one option and the program determines the winner based on predefined game rules.
* This project helped strengthen my understanding of Python fundamentals such as functions, lists, dictionaries, randomness, and conditional logic.



**🎮 Game Rules**



* The game follows these rules:



🐍 Snake drinks Water → Snake Wins

🌊 Water drowns Gun → Water Wins

🔫 Gun kills Snake → Gun Wins



* If both the user and system choose the same option, the game results in a Tie.



**🚀 Features**



✅ Interactive command line game

✅ Random system choice using Python's random module

✅ Game logic implemented using functions

✅ User-friendly output with emojis

✅ Simulated delay using time module for better experience



**🛠️ Technologies Used**



* 🐍 Python
* 🎲 Random Module
* ⏳ Time Module
